## Quick Examples

### Submit a Task via CLI
```bash
sle think "Research Python async patterns"
```

### Submit a Serious Task
```bash
sle think "Implement caching layer" -p backend
```

### Check Status
```bash
sle check
```

### View Report
```bash
sle report
```
